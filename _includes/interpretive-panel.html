{% comment %} interpretive-panel.html Side panel listing the tropes present in a document with toggle checkboxes. Usage: {% assign document_tropes = page.tropes | split: "; " %} {% include interpretive-panel.html document_tropes=document_tropes %} document_tropes: array of trope label strings (split from the CSV "tropes" field). Matching is done against site.data.tropes label values. The panel starts hidden. Show it from outside with: document.getElementById('interpretive-panel').hidden = false; Checking a box adds class trope-active-N to ; unchecking removes it. Requires: _data/tropes.yml and the CSS injected by transcript-viewer.html. {% endcomment %}
